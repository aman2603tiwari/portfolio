# 🎨 Portfolio

A modern, responsive portfolio website built with Next.js and TypeScript to showcase projects, skills, and experience.

**Live Demo:** [https://portfolio-nu-three-95.vercel.app/](https://portfolio-nu-three-95.vercel.app/)

## ✨ Features

- ⚡ **Fast & Performant** - Built with Next.js 15+ for optimized performance
- 🎯 **Fully Responsive** - Mobile-first design that works on all devices
- 🌙 **Dark Mode Support** - Toggle between light and dark themes
- 📱 **Mobile Optimized** - Seamless experience on phones, tablets, and desktops
- ⚙️ **Modern Stack** - TypeScript, React, and latest web technologies
- 🚀 **Easy Deployment** - Deployed on Vercel with automatic deployments

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aman2603tiwari/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

The page will auto-update as you edit files. Hot Module Replacement (HMR) is enabled for a smooth development experience.

## 🎨 Customization

To customize your portfolio:

1. **Edit the homepage** - Modify `app/page.tsx`
2. **Update content** - Edit components in the `components/` directory
3. **Change styles** - Update CSS in `styles/` or component-level styles
4. **Add new pages** - Create new files in the `app/` directory
